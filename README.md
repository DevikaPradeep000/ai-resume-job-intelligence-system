# AI Resume Job Intelligence System

An AI-based application that compares resumes with multiple company
job descriptions to determine eligibility and identify missing skills.

## Objective
To help candidates and hiring teams analyze skill gaps using NLP
and machine learning techniques.

## Features
- Resume parsing
- Job description aggregation
- Skill matching
- Missing skill identification

## Tech Stack
- Python
- NLP
- Machine Learning
